עמוד נחיתה אישי בסגנון **פטיפון ריאליסטי** (Technics) ל-DJ **ארז דלאל**.

## מבנה הדף

- **פטיפון** — שלוש שכבות תמונה:

  - טבעת ניאון אדומה זוהרת מתחת לתקליט.

```bash

# גלשו אל http://localhost:8000
```

## התאמה אישית
- **קישורי רשתות / טלפון / מייל** — ערכו את ה-`href` ב-`index.html` (חיפוש: `yourprofile`, `972525036161`, `erezdalal.com`).
- **מהירות סיבוב התקליט** — המשתנה `--spin-speed` בראש `styles.css` (ברירת מחדל `4.5s`).
- **מיקום/גודל התקליט והכפתורים** — משתני ה-geometry בראש `styles.css` (`--disk-*`, `--icon-*`, `--bN-*`), יחסית לרוחב בסיס של 900px.
- **צבעים** — טוקנים תחת `:root` (`--c-neon`, `--c-accent` וכו').
