# ארז דלל — DJ Profile

עמוד פרופיל אישי (Landing Page) לתקליטן **ארז דלל**, בסגנון House / Mainstream.
האתר בעברית עם תמיכה מלאה ב-RTL, רספונסיבי, ללא תלויות חיצוניות.

## תוכן העמוד
- **Hero** — שם, סלוגן וקריאה לפעולה.
- **אודות** — ביוגרפיה קצרה ותגיות סגנון.
- **שירותים** — חתונות, אירועי חברה, מסיבות פרטיות, הגברה ותאורה.
- **גלריה** — רשת תמונות (כרגע אימוג'ים — להחלפה בתמונות אמיתיות).
- **אירועים** — רשימת אירועים קרובים.
- **צור קשר** — טופס ליצירת קשר וקישורים לרשתות חברתיות.

## הרצה מקומית
פשוט פותחים את `index.html` בדפדפן, או:

```bash
python3 -m http.server 8000
# גלשו אל http://localhost:8000
```

## התאמה אישית
- **תמונות:** הוסיפו קבצים לתיקיית `assets/` והחליפו את ה-`.gallery__item` וה-`.about__photo` ב-`<img>`.
- **צבעים:** משתני העיצוב נמצאים בראש `styles.css` תחת `:root`.
- **תוכן:** כל הטקסטים, האירועים וקישורי הרשתות נמצאים ב-`index.html`.
- **טופס:** כרגע הטופס הוא דמו צד-לקוח. לחיבור אמיתי השתמשו ב-Formspree / Netlify Forms / EmailJS.

## פריסה (Deploy)
ניתן לפרוס בקלות ב-GitHub Pages, Netlify או Vercel — אתר סטטי ללא צורך ב-build.
